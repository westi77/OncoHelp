# Сценарии тестирования

1. История болезни с лечением, полностью соответствующим рекомендациям.
2. История болезни с частичным соответствием лечения.
3. История болезни с явными нарушениями клинических стандартов.
4. История болезни с недостаточными данными для анализа.
5. Проверка устойчивости к различным форматам документов.
6. Проверка диалогового режима после основного анализа.
