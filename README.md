# OncoHelp

OncoHelp — прототип AI-ассистента для проверки соответствия лечения онкологических пациентов клиническим рекомендациям.

Проект разработан в рамках кейс-чемпионата «Инновационные решения для онкологии».

## Назначение
Система анализирует историю болезни онкологического пациента и проверяет соответствие прошлого и текущего лечения клиническим рекомендациям:
- Министерства здравоохранения Российской Федерации;
- RUSSCO (Российское общество клинической онкологии).

Анализ проводится с учётом особых клинических групп, если соответствующая информация присутствует в истории болезни:
- пожилые пациенты;
- беременные пациентки.

Поддерживаются два режима:
- профессиональный режим для врача;
- упрощённый режим для пациента.

## Ограничения
- система не является медицинским изделием;
- не принимает клинических решений;
- не назначает и не изменяет лечение;
- используется исключительно для анализа и пояснения информации.
