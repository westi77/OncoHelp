# OncoHelp

OncoHelp — прототип AI-ассистента для проверки соответствия лечения онкологических пациентов клиническим рекомендациям Минздрава Российской Федерации.

Проект разработан в рамках кейс-чемпионата «Инновационные решения для онкологии».

## Краткое описание
OncoHelp анализирует историю болезни онкологического пациента и проверяет соответствие прошлого и текущего лечения клиническим рекомендациям Минздрава РФ для заданной нозологии.

Система поддерживает два режима работы:
- для врача (профессиональный отчёт);
- для пациента (упрощённое объяснение).

## Статус проекта
Прототип.

## Ограничения
- Система не является медицинским изделием.
- Не принимает клинических решений.
- Не назначает лечение.
- Используется только для анализа и пояснения.
